# Changelog

## 0.3.0 - 2026-06-19

- 增加 CSV 批量目标清单、单进程顺序执行和全局限速复用。
- 增加批次/任务持久化、失败汇总、JSON 摘要和中断恢复。
- 数据库升级到 schema v3，并支持 v1→v2→v3 链式迁移。
- 增加可选 PyArrow Parquet 导出和独立 CI 验证。
- 增加 Windows 任务计划使用说明。
- 项目采用 MIT License。

## 0.2.0 - 2026-06-18

- 引入 `SourceAdapter` 和类型化响应模型。
- 增加视频元数据、分P和视频互动指标观测。
- 增加时间排序及可选完整楼中楼。
- 增加评论父子关系、置顶、状态和历史观测。
- 增加 SQLite v1→v2 事务化迁移。
- 增加热门游标过期重启、`Retry-After` 和结构化请求日志。
- 增加多实体导出、`inspect` 命令、脱敏契约测试和 GitHub Actions。

## 0.1.0 - 2026-06-18

- 初始匿名一级评论采集器。
- 支持热门分页、SQLite 续抓和 CSV/JSONL 导出。
