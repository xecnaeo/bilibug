# Changelog

## 0.7.0 - 2026-06-20

- 增加五个固定生命周期阶段的一级评论主题演化分析。
- 增加阶段TF-IDF关键词、归一化热力表、新增高权重词项和持续主题。
- 阶段至少20条评论且阶段成熟时才参与比较，避免小样本结论。
- 使用全视频单词评论黑名单，防止阶段分析重新暴露短评论原文。
- 拆分报告渲染模块，`report.py` 保留数据查询、分析编排和页面总装配。

## 0.6.0 - 2026-06-19

- 报告核心改为以视频发布时间为零点的一级评论生命周期。
- 增加发布后小时/日评论量、`T50/T80/T90`、峰值和长尾指标。
- 增加基于共同成熟窗口的多视频归一化生命周期比较。
- 楼中楼覆盖不低于90%且存在完整运行时，增加讨论迁移、首次回复延迟和高互动线程集中度。
- 覆盖不足时禁止生成迁移结论，现有采集点互动趋势降为折叠辅助区。

## 0.5.0 - 2026-06-19

- 报告增加可选的一级评论中文词频、TF-IDF 和关键词共现分析。
- 内容分析惰性加载 jieba，只输出聚合词项，不展示评论原文或作者标识。
- 互动趋势默认使用以最新观测为终点的最近 7 天窗口，并增加内嵌 SVG 折线。
- 对少于 3 个点、不足 24 小时和超过 36 小时的数据缺口给出明确提示。
- CI 增加可选内容分析依赖的独立测试任务。

## 0.4.0 - 2026-06-19

- 增加可复用的单文件离线 HTML 报告，覆盖数据质量、互动观测、评论分布和运行诊断。
- 对少于 3 个观测点或不足 24 小时的序列明确标记样本不足。
- 报告只输出聚合统计，不包含评论正文、昵称、用户 ID 或本地清单路径。
- 隔离本地实测数据，并修正批处理 CLI 测试的摘要输出路径。

## 0.3.0 - 2026-06-19

- 增加 CSV 批量目标清单、单进程顺序执行和全局限速复用。
- 增加批次/任务持久化、失败汇总、JSON 摘要和中断恢复。
- 数据库升级到 schema v3，并支持 v1→v2→v3 链式迁移。
- 增加可选 PyArrow Parquet 导出和独立 CI 验证。
- 增加 Windows 任务计划使用说明。
- 项目采用 MIT License。

## 0.2.0 - 2026-06-18

- 引入 `SourceAdapter` 和类型化响应模型。
- 增加视频元数据、分P和视频互动指标观测。
- 增加时间排序及可选完整楼中楼。
- 增加评论父子关系、置顶、状态和历史观测。
- 增加 SQLite v1→v2 事务化迁移。
- 增加热门游标过期重启、`Retry-After` 和结构化请求日志。
- 增加多实体导出、`inspect` 命令、脱敏契约测试和 GitHub Actions。

## 0.1.0 - 2026-06-18

- 初始匿名一级评论采集器。
- 支持热门分页、SQLite 续抓和 CSV/JSONL 导出。
